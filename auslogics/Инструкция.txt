Тестовый хост был auslogics.local
Файл index.php - содержит ссылки на все задания.
Файл check_test_1 - применял для проверки 1-го задания с другого хоста.

test_1.php - Задание 1 - Для проверки нужно использовать ссылку вида http://auslogics.local/files/myfile.exe, при условии что хост: http://auslogics.local.

test_2.php - Задание 2 - Размерность массива динамична, т.к. во первых система по умолчанию каждый раз генерирует новую размерность, так и при удалении лишних дубликатов

test_3.html - Задание 3 - просто страничка, просто стили...

test_4.php - Задание 4. Для защиты данных применялись методы крипто-шифрования openssl, ключём выступает роль почтовый адрес, а вектор шифрования генерируется автоматически и сохраняется в базе. Итог: получив файлы у злоумышленника не будет ключей доступа к персональным данным, даже если он получив базу данных увидит векторы шифрования.
Идея защиты заключается в том, что пользователь, который ввёл почтовый ящик тем самым задал ключ защиты, но в базе данных он уже будет зашифрован =)
Глобальные переменные применялись для отражения статусов на форме, и для применения одного вектора шифрования для нескольких полей.

На всякий случай:

Apache/2.4.23 (Win32) OpenSSL/1.0.2h PHP/5.6.24.
